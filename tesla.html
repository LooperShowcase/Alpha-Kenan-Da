TESLA MODELS

tesla model s

The Tesla Model S is a battery-powered liftback car serving as the flagship model of Tesla, Inc. The Model S features a dual-motor, all-wheel drive layout, although earlier versions featured a rear-motor and rear-wheel drive layout.

tesla model x

Tesla Model X is a battery electric mid-size luxury crossover SUV produced by Tesla, Inc. since 2015. Developed from the full-sized sedan platform of the Tesla Model S, the vehicle is notable in that it uses falcon-wing doors for passenger access.

tesla model Y

The Tesla Model Y is a battery electric compact crossover manufactured by Tesla, Inc. Unveiled in March 2019,[3] it started production at its Fremont plant in January 2020,[4] and started deliveries on March 13, 2020.[5]
